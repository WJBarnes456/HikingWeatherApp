# HikingWeatherApp
Hi-Fi Prototype for University of Cambridge Interaction Design 2017-18, Group 02

Libraries used:
 - GMapsFX
 - JSON.org JSON library

Setup instructions:
 - Duplicate APIKey_Example, rename it to APIKey, and put your key files into it.
 - DarkSky API Key obtained by signing up at "https://darksky.net/dev/docs"
 - GoogleMaps API Key obtainable by "https://cloud.google.com/maps-platform/"
 - Just compile and run!
